# Algorithmic_Palate
# Определение качества вина методами машинного обучения

### Описание проекта

В рамках данного мини-проекта мы выступим в роли инженера-сомелье: нам предстоит определять качество вина с помощью привлечения методов классификации.

---

### Бизнес-задача:

Разработать и оценить качество автоматизированного процесса оценки качества вина на основе его характеристик.

### Техническая задача для специалиста в *Data Science*:

* Провести анализ предоставленных данных о винах и их экспертных оценках качества.

* Разработать протестировать и оценить полученную точность (метрику `accuracy`) в предсказании качества вин *ML*-моделью по сравнению с человеком.

---

### Организационная информация

Ход работы над мини-проектом был разделён на соответствующие содержанию этапы:

* **Чтение и работа с данными**

* **Визуальный анализ данных**

  * **Оценка влияния факторов на качество вина**

  * **Матрица корреляции признаков с целевой переменной**

* **Предобработка данных перед обучением**

* **SCV-модель**

  * **Базовая модель**

  * **Модель с подобранными гиперпараметрами**

* **Итоговый выбор модели**

### Набор данных

[Набор данных](https://drive.google.com/file/d/1uS3vgB9UDT-LgG-sZLaeVNEyA1W5gJwm/view?usp=drive_link) предоставленный для работы содержит информацию о красных винах и их составе. 

Целевой переменной является столбец `'quality'` — это метрика качества вина, составленная экспертом по шкале от 3 до 8.

Познакомимся поближе с нашими данными:

| Наименование столбца | Описание | Пример значения в столбце |
| --- | --- | --- |
| fixed acidity | Уровень естественной, стабильной кислотности вина. | 7.4 |
| volatile acidity | Уровень кислот в вине, которые можно почувствовать носом. | 0.70 |
| citric acid | Уровень лимонной кислоты в вине. | 0.04 |
| residual sugar | Уровень натуральных виноградных сахаров, которые не были преобразованы дрожжами в алкоголь. | 1.9 |
| chlorides | Уровень хлоридов в вине. | 0.076 |
| free sulfur dioxide | Показатель уровня консерванта и антиоксиданта используемого в пищевой промышленности. | 11.0 |
| total sulfur dioxide | Сумма свободного и связанного диоксида серы в вине. | 34.0 |
| density | Показатель плотности вина. | 0.9978 |
| pH | Мера кислотности или щелочности водного раствора. | 3.51 |
| sulphates | Показатель солей и сложных эфиров серной кислоты, широко распространенных в промышленности (они же минералы). | 0.56 |
| alcohol | Уровень алкоголя в вине. | 9.4 |
| quality | Целевая переменная, характеризующая оценку качества вина. | 5 |
